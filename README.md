## Features Implemented
In the backend the features implemented are 
1. GET    /memes
2. POST   /memes
3. PUT    /memes/<id>
4. PATCH  /memes/<id>
5. DELETE /memes/<id>

In the frontend, we have 3 main pages 
1. Submit meme page
2. All memes page 
    1. Dark mode
    1. Edit meme
3. Search meme 
    1. according to person submitted or meme caption
    1. link to each meme
